What is RethinkDB?

  1. NoSQL database to store JSON documents.
  2. Chainable queries written at the programming level & executed serverside. No more query syntax to remember!
  3. Fully featured queries supporting joins, map/reduce, & cursors
  4. Simplified sharding and replication with multiple servers
  5. Includes sexy web interface out of the box
  6. Fairly new to the block. Check the Stability Report.

    3. Write settings on instance config
    4. Write durability at query
    5. No-reply at query

Security

Read the docs article regarding security. TLDR:

  1. RethinkDB Admin Web Panel
    • Simple Reverse proxy + basic auth
  2. Driver Port - what your program connects to
    • Secure connection with `auth` password
    • IPtables / firewall
  3. Cluster Port - how your clusters communicate
    • IPtables / firewall

Features I didn't cover:
